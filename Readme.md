

Άσκηση 2.3
Στο αρχείο q3.c υπάρχει:
Η υλοποίηση του αλγορίθμου merge sort με την συνάρτηση merge(),
η χρήση του με σειριακή εκτέλεση στη συνάρτηση mergeSort_serial()
και με παράλληλη με την mergeSort_parallel() μέσω OpenMP.

Επίσης υπάρχει η συνάρτηση is_sorted() που ελέγχει αν ένα array είναι ταξινομημένο
και χρησιμοποιείται στην main() που εκτελεί τις παραπάνω συναρτήσεις με βάση την
είσοδο του χρήστη.

Συγκεκριμένα για την εκτέλεση της main() πρώτα μεταγλωττίζουμε το αρχείο q3.c μέσω του
Makefile.
Άρα τρέχουμε make

Στη συνέχεια τρέχουμε ./q3 <n> <parallel/serial> <t> όπου
n είναι το μέγεθος που θέλουμε να έχει ο πίνακας.
στο δεύτερο όρισμα γράφουμε parallel ή serial αναλόγως με το αν θέλουμε να τρέξουμε την
παράλληλη ή την σειριακή εκτέλεση.
t είναι το πλήθος των threads που θέλουμε για την παράλληλη εκτέλεση, στην σειριακή δεν έχει
σημασία.

Με την εντολή make clean μπορουμε να σβήσουμε το εκτελέσιμο όταν δεν το χρειαζόμαστε πια.
